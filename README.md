# red_cross
Redcross v2


## Local setup

1. Clone the repository (or fork it, then clone it. Contribute!)
2. Go to the folder install the required gems

```
$ cd red_cross
$ gem install bunlder
$ bundle
$ rake db:create
$ rake db:migrate
$ rake db:seed
$ bundle exec rails server
```

## Deployment

This app runs on heroku.
